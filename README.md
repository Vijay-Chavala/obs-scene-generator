# OBS Lyrics Scene Generator

A frontend-only **Next.js** app that converts a lyrics `.txt` file into an **OBS Scene Collection JSON** that can be imported into OBS Studio.

## Setup

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Then open the URL shown in the terminal (usually `http://localhost:3000`).

## Usage

1. Upload a lyrics `.txt` file.
2. Configure font/layout settings (optional).
3. Generate and download the OBS Scene Collection JSON.
4. Import the JSON in OBS Studio (Scene Collection import).

## Notes

- Input format details: `OBS_Lyrics_Generator_NextJS_Spec.md`
- OBS JSON template: `templates/obsTemplate.json`
